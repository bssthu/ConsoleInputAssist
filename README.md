ConsoleInputAssist
==================

辅助输入，帮助调试需要输入大量信息，但输入格式不统一的程序。

###操作说明

单击“捕获窗口”按钮，然后单击要操纵的窗口。

在TextBox中依次输入要添加的项，单击“添加”或按回车。

在ListBox中选择要发送的项，双击或右键单击或单击“发送”按钮，将内容发到目标窗口。
